# MP3 Player 
A simple MP3 Player built using Python and Tkinter with Pygame for audio playback.

## Features
- Play, Pause, Stop music
- Song list display
- GUI interface using Tkinter

## Technologies Used
- Python
- Tkinter
- Pygame
- Pillow (PIL)

## How to Run
1. Clone the repository
2. Install required libraries
3. Run main.py

## Note
Add your 'Music' folder
Add your MP3 files inside the `Music` folder before running.
